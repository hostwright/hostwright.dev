---

## Host

| Requirement | Supported |
| --- | --- |
| CPU | Apple silicon (arm64). |
| macOS | macOS 26 or newer. |
| Swift | Tools version 6.2. The exact compiler is recorded per run. |
| Apple `container` | Verified against 1.0.0 observation shapes and the narrow mutation surface. |

## Interface versions

| Interface | Version |
| --- | ---: |
| Manifest | 2 |
| Control API | 2 |
| Runtime Provider API | 2 |
| Plugin ABI | 1 |
| State schema | 7 |

The manifest parser accepts `version: 2` and provides a deterministic preview
for legacy v1 and versionless input.

Unsupported versions, fields, endpoints, and clients fail with an explicit
error rather than partial behavior. Container images must provide a
`linux/arm64` variant.

For the full set of supported and unsupported capabilities, see
[Limitations](/reference/limitations/).
