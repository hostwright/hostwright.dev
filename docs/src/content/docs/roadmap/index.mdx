## Locked foundations

- Manifest v2, Control API v2, Runtime Provider API v2, plugin ABI v1, state schema v7.
- Hostwright UUIDs are primary identity; Apple names are attributes.
- One project generation binds to one mutation provider.
- Every mutation uses durable intent, fencing, checkpoints, verification, and safe compensation.
- SQLite is standalone/node-local state; managed etcd supplies cluster consensus.
- Kubernetes pods use a real Hostwright pod-sandbox VM/guest-agent boundary.
- WASI is the default plugin model; signed XPC is reserved for native Apple capabilities.
- Host-native Metal/Core ML/MLX is the accelerator fallback when public guest passthrough is absent.

## Qualification targets

- 100 Macs and 10,000 declared workloads per cluster.
- 1,000 pending workloads across 100 nodes scheduled under one second p95.
- 10,000 local lifecycle cycles and a 72-hour single-host soak.
- Physical three/five-Mac fault matrices and seven-day mixed-fault soak.
- 24 aggregate fuzz CPU-hours for every critical parser/protocol.
- Two complete clean release-candidate runs and zero unresolved P0/P1 defects.
